# Client User Manual

## Overview
This manual guides clients on using E-Legal Connect, including tracking cases, uploading documents, and communicating with lawyers.

## Features
- Dashboard overview
- Viewing case status and progress
- Uploading and sharing documents
- Secure communication with lawyers

## Step-by-step Instructions
1. Log in with your credentials.
2. Use the dashboard to view your cases and their status.
3. Upload documents via the Documents section.
4. Chat with your lawyer using the chat feature.

## Troubleshooting
- For login issues, contact support.
- For document upload errors, check file size and format.

## Help
