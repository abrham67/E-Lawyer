# Judge User Manual

## Overview
This manual guides judges on using E-Legal Connect, including viewing court schedules, managing sessions, and reviewing case files.

## Features
- Dashboard overview
- Viewing and managing court sessions
- Reviewing case files and documents

## Step-by-step Instructions
1. Log in with your credentials.
2. Use the dashboard to view court schedules and sessions.
3. Review case files and documents as needed.

## Troubleshooting
- For login issues, contact support.
