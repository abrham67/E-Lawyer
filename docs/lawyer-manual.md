# Lawyer User Manual

## Overview
This manual guides lawyers on using E-Legal Connect, including case management, document upload, virtual court, and profile management.

## Features
- Dashboard overview
- Managing cases
- Uploading and sharing documents
- Attending virtual court sessions
- Communicating with clients
- Managing your profile and credentials

## Step-by-step Instructions
1. Log in with your credentials.
2. Use the dashboard to view and manage your cases.
3. Upload documents via the Documents section.
4. Join virtual court sessions from the Sessions section.
5. Edit your profile and upload credentials from the Profile page.

## Troubleshooting
- For login issues, contact support.
- For document upload errors, check file size and format.
